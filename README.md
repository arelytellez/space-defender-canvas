# 🌌 Galaxy Star Defender

Videojuego desarrollado con **HTML5 Canvas API, JavaScript y Bootstrap**, como parte de la materia *Introducción a la Graficación 2D e Interfaces de Usuario*.

---

## 🎮 Descripción

Galaxy Star Defender es un videojuego estilo arcade con temática espacial, donde el jugador debe eliminar estrellas que caen desde la parte superior del escenario antes de que lleguen al límite inferior.

El juego integra animaciones dinámicas, efectos visuales galácticos, sonidos personalizados, niveles progresivos y almacenamiento de récord mediante LocalStorage.

---

## 🚀 Características Principales

- ⭐ Generación aleatoria de múltiples objetos 2D
- 🎯 Eliminación de estrellas mediante clic del mouse
- 📈 Sistema de niveles con incremento de dificultad
- 🏆 Registro de puntuación y récord permanente
- 💾 Almacenamiento del high score con LocalStorage
- 🎨 Fondo espacial con diseño galáctico
- 🖱 Cursor personalizado
- 🔊 Sonidos diferenciados (eliminación y pérdida de vida)
- 💥 Efectos visuales al destruir estrellas
- 🌌 Pantalla de Game Over con estilo sci-fi

---

## 🎮 Cómo Jugar

1. Presiona el botón **Comenzar Juego**.
2. Haz clic sobre las estrellas antes de que lleguen al borde inferior.
3. Cada estrella eliminada suma puntos.
4. Si una estrella toca el límite inferior, pierdes una vida.
5. La dificultad aumenta conforme avanzas de nivel.
6. El juego termina cuando pierdes todas tus vidas.
